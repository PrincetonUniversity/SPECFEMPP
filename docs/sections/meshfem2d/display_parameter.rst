
Display parameters
==================

``output_grid_Gnuplot``
~~~~~~~~~~~~~~~~~~~~~~~

Output grid as a Gnuplot file

:Type: ``logical``

.. code-block::
    :caption: Example

    output_grid_Gnuplot = .true.

``output_grid_ASCII``
~~~~~~~~~~~~~~~~~~~~~

Output grid as an ASCII file

Type
    ``logical``

.. code-block::
    :caption: Example

    output_grid_ASCII = .true.
